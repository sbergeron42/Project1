# VR Inventory Pro

A web-based inventory management system for tracking fullbody VR controllers across multiple warehouse locations.

---

## Overview

VR Inventory Pro is a full-stack warehouse and inventory management application designed specifically for VR/motion tracking equipment. The system provides real-time capacity monitoring, inventory tracking, and comprehensive reporting capabilities.

---

## Features

- **Dashboard** — Quick overview of system statistics, alerts, and recent activity

- **Warehouse Management** — Create, edit, and delete warehouses with capacity tracking

- **Inventory Management** — Add, update, transfer, and remove inventory items across warehouses

- **Product Catalog** — View all available products with images and specifications

- **Capacity Reports** — Track warehouse utilization trends over time

- **Search & Filter** — Quickly find warehouses, inventory items, and products

- **Activity Logging** — Automatic tracking of all system changes

- **Report Generation** — Store warehouse capacity snapshots and analyze trends across time

---

## Tech Stack

### Backend
- Java with Spring Boot
- RESTful API architecture
- JPA/Hibernate for database queries
- PostgreSQL database

### Frontend
- Vanilla JavaScript
- Bootstrap 5.3.8 for UI components
- Responsive design for mobile and desktop
